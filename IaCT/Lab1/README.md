# Лабораторная работа №1: Факторизация составного числа

Данный проект представляет собой приложение с графическим интерфейсом на C++ и QML для факторизации составных чисел. Выполнен вариант №2: **$(p-1)$-факторизация Полларда**.

## Описание программы
Программа предназначена для разложения заданного целого числа на простые множители. Алгоритм работает следующим образом:
1. **Тест на простоту:** Сначала введенное число проверяется на простоту с помощью теста Лемана (20 итераций). Если число является вероятностно простым, факторизация отменяется.
2. **Факторизация Полларда $(p-1)$:** Если число составное, применяется метод Полларда для поиска одного нетривиального делителя. 
3. **Рекурсивное разложение:** Найденные делители рекурсивно разбиваются до тех пор, пока не останутся только простые числа.
4. **Вывод результата:** Простые делители группируются по степеням (например, `2^3 * 5 * 11^2`). Также программа выводит затраченное на вычисления время (в миллисекундах) и количество математических итераций.

Ввод чисел поддерживается как вручную через текстовое поле, так и путем чтения из текстового файла. В математическом бэкенде используется `__int128` для предотвращения переполнений при возведении в степень больших чисел.

## Стек технологий
* **Язык программирования:** C++20
* **Графический интерфейс:** Qt 6 (QML / Qt Quick Controls 2, Material Style)
* **Сборка:** CMake

## How to install
### Linux/Mac
```bash
mkdir build
cd build
cmake ../src
make
./iact1
```
